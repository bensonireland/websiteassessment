3D Model Making Process


To make each fruit in blender, I started with a cylinder, and modelled it into the fruit by basing it of an image from the internet.

Firtst I copied an image of the fruit, or bowl, into blender. Then I created a cylinder infront of the image.

Then I shaped the cylinder to match the shape of the object. Making the object a wire frame made this easier.

I did this my moving, rotating, scaling, and extruding the ebjects in edit mode. I also added modifiers and materials to the objects to make them look more realistic and to give them colour.

Example of using edit mode to model the orange based off an image.

Using the wireframe mode made it easier to model against the image.

The final object matches the original image.
